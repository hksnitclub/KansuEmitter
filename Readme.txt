- KansuEmitter 取扱説明書 -
この度は、KansuEmitterをダウンロードしていただきありがとうございます。
このアプリケーションは、パスを入力欄に入力し、テキストファイルに出力するかクリップボードにコピーするか選択して「出力」ボタンを押すとクリップボードに構文がコピーされるアプリケーションです。
ぜひあなた方のプログラミングや教育のご一助になれば幸いです。
