- KansuEmitter 取扱説明書 -
この度は、KansuEmitterをダウンロードしていただきありがとうございます。
このアプリケーションは、パスを入力欄に入力し、「出力」ボタンを押すとクリップボードに構文がコピーされるアプリケーションです。
ぜひあなた方のプログラミングのご一助になれば幸いです。
